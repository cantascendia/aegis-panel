# 怎么从微信熟人圈拉到第一批 5 个客户

> 给"不懂从哪拉客户"的运营者。
> 你刚上线机场,想要 5 个真付费客户证明产品有市场。
> 答案不是 TG 不是论坛 — 是你微信里的 30-50 个老朋友。
>
> 配 `docs/ai-cto/ROADMAP-launch-B-to-C.md` Week 3 段。
> 本指南是 2026-05-01 用户实操总结,Day-by-Day 跟进。
> 与 `RECRUIT-COPY.md` 的关系:那份是 6 渠道通用文案池,这份是微信单渠道的实战节奏 + 心理学。

---

## 为什么从熟人开始(不是公域投广告)

### 数学

- 公域 TG 投广告 — 100 曝光 → 1 点击 → 0.1 试用 → 0.02 付费(转化 0.02%)
- 熟人微信 — 50 名单 → 30 私聊 → 10 试用 → 3-5 付费(转化 6-10%)

差距 300x。**熟人是创业第一波客户的王道**。

### 心理学

- 熟人对你"产品"持怀疑,但对"你"是信任的(知道你不会跑路)
- 你不是在卖产品,是在帮朋友解决一个具体问题(YouTube 看不了 / ChatGPT 慢 / 出差访问公司 Slack)
- 熟人即使试用不付费,也不会拉黑你 — 公域用户会
- 熟人付费后给你的反馈是真话,不是评论区表演 — 这是产品打磨的金矿

### 但有禁区

- ❌ 不要朋友圈广告(显得低端,损长期信任,圈子里传开你"开始卖货了")
- ❌ 不要群发("亲爱的朋友 福利来啦"这种)
- ❌ 不要在群里 @ 所有人推销(会被踢)
- ❌ 不要在工作群 / 同学大群发(职场 / 关系链复杂,慎用)
- ✅ **一对一私聊**,根据对方 profile 调话术
- ✅ **看到对方近期朋友圈相关动态再切入**(自然,不像扫荡)

## Day 1:列你的"潜在客户名单"(30 分钟)

### 打开微信通讯录,标记符合任一条件的朋友

#### 一定能成的(优先列)

- [ ] **程序员 / 设计师**:每天用 GitHub / Figma / Discord / Stack Overflow / npm
- [ ] **跨国公司 / 海外团队工作**:每天 Zoom / Slack / Notion 国际版 / Google Workspace
- [ ] **留学生 / 海外党 / 海归**:回国发现 YouTube / Google / Gmail 都看不了
- [ ] **出差党 / 商务**:经常跨境,需要随地访问公司国际服务

#### 中等概率(也列)

- [ ] **自媒体 / 写手 / 翻译**:经常用 ChatGPT / Claude / Notion AI / DeepL
- [ ] **游戏玩家**:玩 Steam 国际服 / 美区 PSN / Switch eShop / 国际版手游
- [ ] **音乐 / 电影爱好者**:Spotify / Apple Music 国际版 / Netflix / Disney+
- [ ] **学术 / 科研人员**:Google Scholar / arXiv / 国外期刊 / Wikipedia

#### 低概率(放后面试)

- [ ] **被坑过的机场用户**:你听过他抱怨"XX 机场跑路了" / "今天又卡了"
- [ ] **担心账号被盗**:用过 1Password / 关心隐私安全
- [ ] **加密货币玩家**:用 USDT / Binance / Coinbase

#### 不要列(转化率太低,徒增尴尬)

- ❌ 长辈 / 家人(他们用得上但解释成本高,而且不愿付钱给晚辈)
- ❌ 现任老板 / 直属同事(职场关系微妙,不要让"赚朋友钱"和"工作"混)
- ❌ 多年没联系的(突然来推销很怪,先聊几次再说)
- ❌ 普通朋友(没明确翻墙需求,你强推让对方为难)
- ❌ 关系正在僵的(别用商业关系催化矛盾)

### 目标

- 列出 **30-50 个名字**(写在备忘录 / Excel / Notion)
- 转化率 **6-10%** 算正常,12% 已经是上限
- 期望出 **3-5 个真付费**(B 阶段第一批)

## Day 2-3:写 5 类话术模板

每个朋友的 profile 不一样,**用同一套话术 = 像群发**。给 5 类各准备一段。

### 模板 1:被坑过的工程师(最易转化)

```
A:[小聊几句,问问他最近,然后]
B:对了你之前不是用 XX 机场被坑过吗,跑路那次?
A:啊对,被坑了 80 块还是多少...
B:我最近自己搞了个翻墙的,跟之前那种不一样 —
   代码全开源放 GitHub 上的(cantascendia/aegis-panel)
   跑路就等于公开打脸,没那个必要
   抗封算法 + Reality 巡检都做了,我自己天天用 + 几个朋友在用
B:[等他问"多少钱" / "稳定吗"]
B:¥30/月 100GB,送你 3 天免费试用先用着,觉得不行直接告诉我
   (你不是那种朋友硬推不好意思的人,直说就行)
```

### 模板 2:留学生 / 海外党

```
A:[随便聊,问他最近回国还是在外]
B:对了你回国看 YouTube / Google 是不是变慢了?
A:对啊,公司 VPN 也不让用 YouTube
B:我最近做了个东西,日本 / 香港节点延迟 30ms 左右,看 4K 流畅
   你的话试用 3 天免费,如果觉得快就续 ¥30/月
B:[等他问什么]
B:我代码全在 GitHub,跑路就背叛 — 反正我是不会跑路的
   要不要给你开?
```

### 模板 3:出差 / 商务党

```
B:你最近不是要去 [国家] 出差吗
A:对啊
B:那边网络...你提前装个翻墙吧,我这边日 / 港 / 美西 / 新加坡都有节点
   到了哪个城市自动切最快的
   我自己做的,3 天免费,觉得行 ¥30/月,流量够你出差用
   (Slack / Notion / Gmail 这些都能稳)
```

### 模板 4:自由职业 / 用 ChatGPT

```
B:你最近不是经常用 ChatGPT 写东西吗?
A:对,但有时候连接不稳定 / 经常掉线
B:我的机场美西硅谷直连,GPT 几乎不卡
   想试就告诉我,3 天免费,不行就拉倒
```

### 模板 5:普通朋友(没明确需求)

**不要主动推**。**等他问** "你这是干嘛 / 最近忙什么" 时再说:

```
A:你最近在搞什么?
B:在做一个翻墙的小工具,自己用着挺好,几个朋友也在用
   你要的话我送你 3 天免费试用,试试看
   (不是推销,如果用不上扔掉就行)
```

**关键**:这一类要**给台阶**,让对方"试试看"零心理压力。

## Day 3-7:私聊节奏

### 一天联系几个?

- 每天 **5-10 个**(超过 = 你状态会假,效果反而差)
- 间歇至少 30 分钟(看起来不像群发)
- 同一周不要找完所有名单 — **分两周慢慢推**,留缓冲应对反馈

### 单个对话节奏

1. **第一句**:聊别的(他最近的朋友圈 / 上次聊天的话题),别上来推销(对方会防御)
2. **3-5 句后**:话题自然引到他的痛点(YouTube / GPT / 出差 / 公司国际版)
3. **看他反应**:
   - 有兴趣 → 继续
   - 不感兴趣 → 立刻撤回("不行就算了,不强推哈")
4. **关单**:不超过 3 段话推介,推了不接受**就停**(逼客户 = 死客户 + 死朋友)

### 死亡话术(千万别说)

- ❌ "限时优惠,今天免费"(像传销)
- ❌ "你要支持我创业"(道德绑架,朋友最反感)
- ❌ "现在不买以后涨价"(廉价营销话术)
- ❌ "我给你算笔账,你一个月省 XX 钱"(他不傻,被算账感觉被当韭菜)
- ❌ "兄弟 / 姐妹给个面子"(关系越近越不能用)

### 黄金话术

- ✅ "试用 3 天免费,觉得不行就算了"(零风险)
- ✅ "我自己天天用,你看效果就知道"(自证 + 现身说法)
- ✅ "不行就告诉我哪不行,我修了再给你看"(开放反馈,显得在意)
- ✅ "用不上就扔掉,不影响咱关系"(主动给台阶)

## Day +1, +3, +7:跟进话术

### D+1(开账号第 2 天)

```
B:试用还顺手吗?有什么不对劲告诉我
```

- 对方说**好用** → 心里 +1,不催不烦
- 对方说**慢 / 断** → 立刻看 dashboard 找原因(节点 / 客户端 / SNI)
- 对方**没回** → 可能没装,等 D+3 再问

### D+3(用了 3 天)

```
B:这两天 YouTube 4K / GPT 看着还行吗?
```

(具体场景问,比"还在用吗"好,得到具体反馈;让对方说"4K 流畅"或"GPT 不卡"比"还行"信息量大 10 倍)

### D+7(试用到期前 1 天)

```
B:3 天试用快到期了,要不要续 ¥30/月 100GB?
   支付宝 / USDT 都行,选个方便的
```

### 关单技巧

- **不催** — "随时,不急,你想清楚再说"
- **给选择** — 不是"买不买",是"支付宝还是 USDT"(已经默认买,只问支付方式)
- **价格透明** — 不要"原价 ¥XX 优惠 ¥XX"鬼把戏,熟人最反感这套
- **不到期就断** — 试用过了不付费就停,不要"再送你 3 天"养懒(但保留账号 30 天,以防他回头)

## 常见问题应对(微信场景版,与 CUSTOMER-FAQ.md 互补)

### Q1: "你这翻墙是不是违法?"

A:技术上 GFW 是中国大陆司法管辖范围,我服务器在境外。AGPL 开源,代码 GitHub 公开,你看到的是工具,不是用法。法律灰色地带,自己拿捏。我自己用,几个朋友也在用,都正常。

### Q2: "你跑路怎么办?"

A:代码在 GitHub 上 cantascendia/aegis-panel,跑路 = 公开打脸,我没必要这么干。我做这个一是自己用,二是有几个朋友在用 — 不是上千用户的大商业。退一万步,代码开源了你也可以自己拉服务器跑(虽然你不会)。

### Q3: "市面上机场便宜的 ¥10/月,你 ¥30 凭什么?"

A:¥10 机场 80% 是转售流量(自己不维护节点),被封了或跑路了你没人找。我自己维护节点,出问题直接微信我修(你看,你都能直接微信我了)。30 块买的是"知道运营是谁"。如果你觉得 ¥10 够用,真心建议你先用着,出事再找我。

### Q4: "我已经用 XX 机场了"

A:那挺好的,我也不想抢生意 — 备一个总没坏处,有时候 XX 机场跟不上,你切到我这。3 天免费,不用就扔。(给台阶,不要硬怼别人在用的)

### Q5: 客户主动问 "我能不能给我朋友也开一个"

A:能。但你的账号是你的,不要给他你的订阅链接(IP 限制会断你两个的连接)。我可以单独给他开一个 — 你介绍来的朋友我送他 3 天试用。(自然引出推荐机制,不要主动卖)

### Q6: "你做这个一个月赚多少?"

A:刚上线,几个朋友试用,主要先把产品做好。不指望靠这个发财,够覆盖服务器成本 + 给自己加菜就行。(别撒谎说"很多人买",别哭穷,平静交代)

### Q7: 朋友试用完不付,但也不明说

A:**不要追**。发一句"账号到期啦,你不续就停了哈,不影响咱关系"。然后停。
- 70% 概率他真的用不上(不是你产品问题)
- 30% 概率他在犹豫,3 个月后他可能又来找你

## 转化漏斗 + 数字管理

### 实际数据预期(假设 30 名单)

| 阶段 | 期望数 | 期望比 |
|---|---|---|
| 名单 | 30 | 100% |
| 私聊到 | 25 | 83% |
| 表示有兴趣 | 12 | 40% |
| 接受试用 | 8 | 27% |
| 试用后表示满意 | 5 | 17% |
| 真付费 | 3-4 | 10-13% |

**3-4 真付费 = B 阶段第一波目标完成**。如果 50 名单跑完只有 0-1 付费,说明话术或定价或产品有问题,**先停下复盘,不要硬扩公域**。

### 失败 = 学习

- 每个**没成**的也记下原因(对方明确说"不需要" / "已经有了" / 没回 / 礼貌拒绝)
- 这 25-27 个人是你以后做产品改进的金矿:
  - "已经有了" → 你的差异化在哪
  - "不需要" → 用户教育还是 ICP 错了
  - "没回" → 话术第一句不够吸引

### 跟踪表(必备)

建议:用 Excel / Notion 列(每行 1 个朋友):

| 名字 | profile 类型 | 私聊日期 | 反应 | 试用 | 试用反馈 | 付费 | 续费 |
|---|---|---|---|---|---|---|---|
| 小张 | 1(工程师) | 05-02 | 感兴趣 | Y(05-03) | "GPT 不卡了" | ¥30(05-06) | - |
| 小李 | 4(自媒体) | 05-02 | 已经有 | - | - | - | - |
| ... | | | | | | | |

每周末更新,看趋势 — 哪一类 profile 转化高就多找同类。

## 最后:第 1 个真付费客户的"正确状态"

不是:**亢奋 + 朋友圈炫耀**(显得没见过钱,而且暴露了你的"商业意图",让没买的朋友尴尬)

是:**告诉自己市场验证了 +1**(产品方向对,继续打磨)

第 1 个客户 = 你产品的"第一个 stake holder"。把他养好,**他会带 3 个朋友**(口碑放大效应)。

第 1 个客户给你的反馈是产品 PMF 的最强信号 — 比 1000 个公域用户的"在吗"更有价值。

## 到 5 个客户后,下一步看 ROADMAP-launch-B-to-C Week 4-5

- 5 客户 → 6-20 客户:上述方法 + 老客户介绍(给老客户 9 折当月费 / 续费天数奖励)
- 20+ 客户:才考虑 TG / 论坛 / 知乎等公域(有信任锚 + 真实评价)

**不要在没把熟人路走完之前去碰公域** — 公域的 0.02% 转化率会让你怀疑人生,而那不是产品问题,是渠道问题。

## 维护

- 每周末跑 `bash scripts/aegis-watchdog.sh` 看 panel 健康
- 每周末看跟踪表,标记下周要跟进的(D+1/D+3/D+7)
