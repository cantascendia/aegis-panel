# 用支付宝买 USDT 付款指南(客户面)

> **预计耗时**:首次 15 分钟,以后 3 分钟
> **难度**:低(只要会用支付宝)
> **适合**:中国大陆客户,只有支付宝 / 微信 / 银行卡,没有 USDT

---

## 为什么要走这一步?

我们不直接收支付宝。原因:

1. 机场行业监管特殊,直接收会有风控冻结风险(无论是商户还是客户都不安全)
2. USDT 链上付款**合法可追溯,反而最稳**
3. 一次学会,以后续费 3 分钟搞定

**钱不经过我们任何账户**,你直接在交易所买 USDT,通过区块链发到收款地址。

---

## Step 1:下载 OKX(推荐,5 分钟)

> 也可用币安 / Bitget,流程一致。OKX 中文界面 + 支付宝 P2P 商家最多,首推。

1. 去 `okx.com`(需翻墙,你已经在用我们服务可直接打开)
2. 下载 App(iOS / Android)或用网页版
3. 注册账号:邮箱 / 手机号都行,推荐邮箱避免国内手机号关联
4. 完成 KYC(身份认证):
   - 大陆身份证 / 护照都接受
   - 上传证件正反面 + 人脸识别
   - 5-30 分钟审核通过

> ⚠️ 不要用国内交易所 App(火币 / 抹茶等),它们对大陆用户有限制。OKX / 币安 / Bitget 是海外注册的。

---

## Step 2:用支付宝买 USDT(P2P,8 分钟)

OKX App 操作:

1. 首页底部点「**交易**」→「**C2C**」(或 P2P)
2. 选「**买入 USDT**」
3. 付款方式选「**支付宝**」(也可选微信 / 银行卡)
4. 输入金额(例:¥30 大概买 4.2 USDT,按当前汇率)
5. 选一个交易商:
   - **优先选订单数 ≥ 1000 + 完成率 ≥ 99%**(老商家稳)
   - 看清单价(¥7.1 ~ ¥7.3 / USDT 浮动)
6. 点「下单」→ 商家显示**收款支付宝账号**
7. 用支付宝**直接付**给那个账号(注意:**不要写"USDT / 数字货币 / 充值"等字样在备注**,只写订单号或留空)
8. 付完点「**已付款**」
9. 等商家确认(通常 1-5 分钟)→ USDT 到你 OKX 账户

> ⚠️ **重要**:支付宝转账备注一定不要出现敏感词。商家通常会提醒你只写"祝您生活愉快"或留空。

---

## Step 3:提币到 Nilou Network 收款地址(2 分钟)

回到 Nilou Network dashboard:

1. 选你要的 plan(月付 / 季付 / 年付)
2. 点「USDT TRC20 付款」→ 显示:
   - **收款地址**(一长串字符,T 开头)
   - **memo / 标签**(重要,带这个钱才能自动到你账户)
   - **金额**(精确到 4 位小数,例:4.1672 USDT)
3. 复制收款地址 + memo

OKX App 操作:

1. 「**资产**」→「**提币**」→ 选「**USDT**」
2. 链选「**TRC20**」(⚠️ 不要选 ERC20,手续费贵 100 倍)
3. 粘贴 Nilou Network 给的**收款地址**
4. **memo / 标签**栏也粘贴(很多客户漏这一步导致到账延迟)
5. 数量填**精确到 4 位小数**的金额
6. 提交 → 短信验证 / Google Authenticator → 确认
7. 5-30 秒到账,Nilou Network panel 自动开通账号

---

## Step 4:验证开通(30 秒)

回 Nilou Network dashboard:

1. 刷新页面
2. 看到「订阅生效,有效期到 YYYY-MM-DD」= 完成
3. 同时收到邮件 / 微信通知(看你绑定方式)

## 常见问题

**Q1:OKX KYC 审核没过?**
- 换护照试,大陆身份证偶尔被拒
- 或者换币安 / Bitget,流程一样

**Q2:支付宝付款被风控?**
- 选「订单数 ≥ 5000」的老商家(他们走过千万流水,风控低)
- 单笔 < ¥3000 不容易触发
- 如果支付宝弹"风险提示",取消订单换商家

**Q3:转账后商家不放币?**
- OKX 有强制仲裁,15 分钟商家不放币 → 申诉 → OKX 介入 → 24 小时内处理
- 不会有"币丢了"的情况,这是 OKX 担保

**Q4:提币 TRC20 / ERC20 选哪个?**
- **TRC20**(波场链):手续费 ¥1-3,5-30 秒到账 ✅
- ERC20(以太坊):手续费 ¥30-100,慢且贵 ❌

**Q5:memo / 标签忘填?**
- 联系运营(微信 / TG),提供你的提币 hash + OKX 截图,人工补开
- 但**下次记得填!**自动到账 vs 人工,区别就这一个字段

**Q6:看不懂英文界面?**
- OKX / 币安 / Bitget 都有简体中文,App 设置里切

**Q7:第一次操作怕出错?**
- 加运营微信 / TG,我陪你走一遍,15 分钟搞定
- 第二次以后你自己 3 分钟搞定

## 为什么这个流程是安全的?

| 环节 | 风险 |
|---|---|
| 你用支付宝付钱给 OKX P2P 商家 | 商家是 OKX 担保的认证商户,有保证金 |
| OKX 给你 USDT | 链上交易,可追溯 |
| 你提币到 Nilou Network | 标准 USDT 转账,合法 |
| Nilou Network 收 USDT 开通 | 链上 SaaS 订阅付款 |

整个链路里,**没有"VPN 商户收支付宝"这种敏感动作**,所以无人会被风控。
