## 维护

每月 review 一次,根据实际客户反馈调整 4 张卡片用词。

跟踪信号:
- 哪张卡片转化率最高(发完 → 试用 → 付费)→ 加大权重
- 哪张卡片最容易引起反问(说明痛点击中)→ 优化表述
- 哪些客户 profile 没覆盖 → 补充新 profile 路径

版本由 LESSONS L-XXX 跟踪(发版前更新本文件 commit message 引用)。

变更触发:
- 价格档变 → 卡 1/卡 4 数字同步
- 新节点上线 → 卡 4 节点列表更新
- 被封率 / 行业均值数据更新 → 卡 1 数字刷新
