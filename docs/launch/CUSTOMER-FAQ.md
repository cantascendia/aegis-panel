## 故障 / 问题

### Q21. 连不上怎么办？
检查顺序（按概率）：
1. App 里确认订阅是否过期 / 流量是否用光
2. 切换 5G/WiFi 重试
3. 设置 → 通用 → 日期与时间 → 自动设置打开（TLS 握手对时间敏感）
4. 重启 App
5. 还不行 → 截图发我

### Q22. 显示 "ERR_SSL_PROTOCOL_ERROR"？
浏览器层报错，常见 3 种：
1. **目标网站对代理 IP 反爬**（如某些银行 / 腾讯系）→ 换站测，能开就是这个站事
2. **DNS 漏出** → App 设置改 DNS 1.1.1.1 或 8.8.8.8
3. **客户端时间错** → 系统时间自动设置打开

### Q23. YouTube 看 4K 卡？
1. 看右下角实际跑多少 Mbps（fast.com）。低于 25Mbps 就不够 4K 了。
2. 高峰期（晚 8-10 点）慢一点正常，错峰看
3. App 里切换"Vision"流（默认是）和"原始"，对比

### Q24. ChatGPT / Claude 用不了？
确认：
1. 节点切到东京（不是中转节点）
2. 浏览器 cookie / cache 清掉，登录 chatgpt.com
3. 还不行：账号被风控了，跟我无关，需要换 OpenAI 账号

### Q25. Netflix Japan / DAZN 解锁？
**真直连日本 IP**，所以 Netflix JP / DAZN / 日服游戏 / 日区 App Store 都能用。Netflix US / 欧洲服需要换节点（暂未提供）。

### Q26. 国内网站打不开了？
App 切到"绕过国内"或"路由模式：分流"，国内站走直连，国外走代理。**不要用全局模式**访问国内站。

### Q27. 打游戏延迟变高了？
代理就是会增加延迟。打国服游戏 → 关掉代理。打日服 / 美服游戏 → 用代理 ping 比直连还低。

### Q28. 这个怎么算法律？
你买的是境外 VPS 服务器的"网络中转访问"。**自用不主动传播无问题**。但如果用来做坏事（批量爬境外数据 / 跑机器人 / 黑产），会被立刻封号且不退款。

### Q29. 服务器宕机怎么办？
- 计划维护：提前 24h 通知
- 突发：我自己手机也用，断了我比谁都急；通常 30 分钟内恢复
- 长期挂掉：自动给所有客户加补偿天数

### Q44. 加密强度？
TLS 1.3 + AES-256-GCM。**国家级 DPI 想破解需要量子计算机**。够用 5-10 年没问题。

### Q45. 端口有限制吗？

### Q49. 跨网络（5G/WiFi 切换）会断吗？
TCP 连接会断重连，约 1-2 秒。VLESS+Reality 重连快，不影响日常使用。
