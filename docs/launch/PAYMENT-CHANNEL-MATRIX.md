## 2. 为什么不直收支付宝

**支付宝官方商户**:
- 国内版需工商执照 + 大陆银行账户 + 类目审核 — VPN 业务 100% 拒
- 国际版(Alipay International)需海外法人 + ICP — 同样不开 VPN 类目

**四方 / 五方聚合**(易支付 / 虎皮椒 / payjs / YunGouOS / BeePay 等):
- 底层接的是个人微信 / 支付宝码池
- 商户层(运营你)依赖聚合方稳定性 — 聚合方被风控 → 你的钱链断 / 资金被扣留
- 2024-2025 监管收紧后,此类聚合方平均存活周期 6-18 个月
- **结论(修订)**:**单通道使用风险高**,但作为副通道+轮换+监控可控。详见 §3 Plan B

**个人收款码 / 朋友代收**:
- 大陆个人卡反洗钱阈值低,大额异常交易直接冻卡
- 帮信罪刑事风险(2024 起执法收紧)
- 不能程序化(无 webhook)
- 不可 scale

### 3.1.3 ⚠️ 重大订正:YunGouOS 个人版 ≠ 第三方聚合方

operator 调研后发现 YunGouOS 个人版**不是传统第三方聚合方四方支付**,而是**微信支付服务商伙伴模式**(由微信认证合作伙伴帮个人申请真实官方商户号)。

#### 实情对比

| 维度 | 传统聚合方假设(虎皮椒 / payjs)| YunGouOS 个人版真相 |
|---|---|---|
| 底层 | 个人收款码池 | **真实微信官方商户号**(2024 年开放个人版)|
| 费率 | 2.4-2.8% | **最低 0.38%**(数字商品/软件类目) |
| 实名前移 | 间接(聚合方挡前) | **直接**(身份证 + 银行卡 + 微信号一致 + 微信官方商户助手小程序)|
| 风控风险 | 中(聚合方依赖)| **低**(直接腾讯,跑路风险消失) |
| 类目审核 | 聚合方包装,卡较松 | **腾讯直审,VPN 类目仍卡** |
| 资金流向 | 经聚合方账户中转 | 直达个人银行卡 |

#### 这意味着什么

YunGouOS 个人版 = **§5 "官方微信商户伪装软件订阅"路径的简化版**(免营业执照 + 服务商代提交):

✅ **优势**(对比聚合方):
- 费率接近官方最低档(0.38%)
- 真官方商户,反诈 / 客户投诉时你是合规商户
- 资金直达,无聚合方跑路风险
- 不需要营业执照(腾讯 2024 年个人版商户改革)

⚠️ **代价**(对比聚合方):
- 实名直接绑定 VPN 业务(冻结后 ≥5 年实名记录)
- 类目过审仍由腾讯做 — 这是**核心卡点**
- 个人版限额:单日 ¥5万 / 单月 ¥30万(0-200 用户阶段够,远期需考虑)
- 通过率重估:网站清洗 + SaaS 包装后 **20-35%**(网站含敏感词 < 5% 必拒)

### 3.6 彩虹易支付"自建"为何不推荐(决策记录)

**是什么**:彩虹易支付 = 中文圈开源支付聚合系统(同源 fork:码支付 / 码云支付 等)。"自建"=你部署系统到自己服务器,自己去对接 N 个下游通道商。

**架构**:
```
客户扫码 → 你部署的彩虹后台 → 多个下游通道(他们底层接个人收款码池) → 钱到下游 → 下游打款给你
```

**为什么不做(0 → 200 用户阶段)**:

| 风险 | 详情 |
|---|---|
| 下游通道才是风险源 | 你只是替自己做"找通道"的工作,**底层风控问题完全没解决** |
| 下游跑路 | 下游收钱不打款给你 — 站长圈常见事故,资金日均滞留 ¥几千-几万 |
| 法律责任前移 | 自建 = 你显式承担"支付服务提供方"角色,反诈警察查到客户付款 → 你是直接代理人,**风险比用虎皮椒更高** |
| 运维成本 | 服务器 + 对账 + 通道维护 + 异常 ≈ 一个全职 sub-agent 工作量 |
| 不省钱 | 下游 1.5-2.5% vs 虎皮椒 2.5-3%,只省 0.5-1%,补不回运维成本 |
| 下游获取门槛 | 通道资源在站长"暗市"流通,新手大概率被坑 |

**何时考虑自建**:年流水 ≥ 100 万 RMB(约 2000+ 付费用户),议价能力 + 运维投入都成立。

**当前(0 用户)做自建 = §32.5 反模式 #1 Vibe Shipping(为不存在的规模做架构)**。

## 4. Plan A:OTC 引导(保底永久通道)

### 工作流

```
客户在 dashboard 选 plan → checkout 显示两条路径:
  路径 A: 我有 USDT → 直接付 TRC20 地址(已 ship)
  路径 B: 我用支付宝 → 跳转「OTC 教程」(本 PR ship)
```

路径 B 教程指引客户:

1. 下载 OKX / 币安 / Bitget App
2. 注册(护照 / 大陆身份证均可,海外交易所 KYC 友好)
3. 充值人民币 → 选"P2P 交易"或"快捷买币"
4. 用支付宝 / 微信 / 银行卡向 P2P 商家买 USDT(TRC20 链)
5. 提币到 Nilou Network 收款地址(panel 给的 memo / 标签自动匹配)
6. 5-30 分钟自动到账

### 商户风险评估

| 风险点 | 评估 |
|---|---|
| 商户账户冻结 | 0(钱不经过你) |
| 反洗钱调查 | 0(USDT 入账,链上可追溯,合法 crypto 交易) |
| 帮信罪 | 0(你提供 SaaS,客户自付 USDT) |
| 资金扣留 | 0(交易所→你钱包,即时) |

### 客户痛点

- ⚠️ **首次客户**:OKX 注册 + KYC + P2P 学习曲线 ~15 分钟
- ⚠️ **小额客户**(¥30 / 月)觉得麻烦
- ✅ **大额 / 长期**(¥80 季度 / ¥240 年度)接受度高

1. 教程必须图文并茂(本 PR 配套 `HOW-TO-BUY-USDT-FROM-ALIPAY.md` ship 中)

4. TG bot 半自动化:`/buy_usdt` 命令推送教程 + 推荐 P2P 商家筛选

- ❌ 不接易支付(纯第三方聚合,2024 后稳定性进一步下降,劣于虎皮椒 / YunGouOS)
- ❌ 不**单通道依赖**虎皮椒 / payjs / YunGouOS(必须双通道并行 + USDT 兜底)
- ❌ 不申请 Stripe / Komoju / PayPal(VPN 类目限制 + 锁号风险)
- ❌ 不用个人微信 / 支付宝码收款(冻卡 + 法律风险)
- ❌ 不让朋友代收(帮信罪 + 友情成本)
- ❌ 不做"代购 USDT"中介服务(把 OTC 风险转回运营)
- ❌ 不在 Plan B 副通道收境外卡 / 大额单笔 ≥ ¥3000(触发风控加速)
