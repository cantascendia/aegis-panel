# 虚拟办公室申请指南(個人事業主隐私保护)

> 状态:Step 4.0(prerequisite of 開業届 / 銀行 / Paddle 申请)
> 适用:在日本 個人事業主,需要事業所所在地 + 電話番号 對外公開但不能泄露個人住址 / 私人手机
> 预计耗时:30 分钟在线办理 + 3 营业日审核
> 关联:`HOW-TO-FILE-KAIGYOU-TODOKE-JP.md`、`PAYMENT-DECISION-FINAL-JP.md`

## 1. 为什么必须用虚拟办公室

### 特定商取引法(特商法)强制公开

日本对所有 e-commerce / 数字订阅业务要求在网站显示"特定商取引法に基づく表記"页,内容**必须**含:

| 项 | 强制公开 | 不能藏 |
|---|---|---|
| 事業者の名前 | ✅ 本名 / 屋号 | — |
| 事業所の住所 | ✅ | ❗ **不能写"請求があった場合に開示"**(2022 消費者庁修订后这条作废) |
| 電話番号 | ✅ | ❗ 必须真实可拨号码 |
| メールアドレス | ✅ | — |

**用個人住址 + 個人手机号** = 你的本名 + 家庭地址 + 私人手机永久公开在网站 + Paddle invoice。

### 风险场景

- 中国客户不开心 → 网搜你住址 → 上门 / 电话骚扰
- 客户被 GFW 抓走 → 反诈警察查 Paddle 流水 → 关联到你住址
- 自然人身份(本名 + 住址 + 电话)与 VPN 业务永久关联 — 即使你将来转行,数据不可消除

## 3. 推荐:GMO オフィスサポート

### 理由

- **¥660/月最便宜**(3 年总成本 ¥23,760 ≈ 200 USD,远低于地址泄漏潜在损失)
- 隶属 GMO 系(GMO Internet 上市公司)— 跑路风险接近 0
- 個人事業主開業 / 法人登記 / 特商法表記 **全部允许**(看契约书「住所利用範囲」)
- 月转送 1 次免费(税务署 / 銀行 / Paddle 邮件不会丢)
- 在线办理,**3 营业日开通**
- 違約金少,1 个月可解約

### 拠点选择(看你住所)

| 你住 | 推荐拠点 | 路程 |
|---|---|---|
| 東京都内 | 渋谷 / 新宿 / 銀座 | 30-45 分钟 |
| 神奈川 | 横浜 / 渋谷 | 30-60 分钟 |
| 関西 | 梅田(大阪)| 30-60 分钟 |
| 愛知 | 名古屋駅前 | 30 分钟 |
| 福岡 | 博多 | 30 分钟 |

**原则**:選**離家 1 小时以内**的,偶尔需要去取实物邮件 / 包裹。

### 不要选

- 都心一等地高端拠点(如丸の内 / 大手町)— 不必要,贵 5-10x
- 离家 > 1 小时的拠点 — 取邮件不便

## 4. GMO オフィスサポート 申请 step-by-step

### Step 4.0.1 進入

打开 `office.gmo.jp` → 顶栏点「お申し込み」

### Step 4.0.2 选 plan

| Plan | 月费 | 含 | 推荐 |
|---|---|---|---|
| **転送なし** | **¥660〜** | 住所利用 + 邮件保管(自取)| ✅ 個人事業主 + 客户少 |
| 月 1 回転送 | ¥1,650〜 | 上 + 月 1 次邮件转送到家 | ⭐ 你客户多 / 离家 > 30 分钟 |
| 週 1 回転送 | ¥2,750〜 | 上 + 週次转送 | 客户多时 |
| **法人登記オプション** | + ¥330〜 | 用作法人 / 個人事業主 開業届 | **必须勾**(¥990/月起) |

**推荐组合**(个人 0-50 用户):
- 月 1 回転送 + 法人登記オプション = **¥1,980/月**
- 入会金 ¥5,500 一次性

### Step 4.0.3 选拠点

按 §3 推荐(離家 1 小时内)。

### Step 4.0.4 在线填表

输入:
- **氏名**:本名(GMO 内部 KYC,**不公开**)
- **生年月日**
- **個人住所**:住民票上的(GMO 内部用,**不公开** — 这是为什么虚拟办公室能保护隐私的关键)
- **メール**:你的工作邮箱(用 nilou.network 域名最好)
- **電話番号**:私人手机(GMO 内部用,**不公开**)
- **業種**:**ソフトウェア / IT サービス**(关键!与開業届 業種一致)
- **業務概要**:同 開業届 §4.2 日文话术(开源软件订阅托管服务)

### Step 4.0.5 身份证 + 信用卡

- 上传 マイナンバーカード(正反面)/ 運転免許証 / パスポート 任一
- 信用卡(月费扣)— Visa / MC / JCB / Amex

### Step 4.0.6 等审核

- 审核 3 营业日(GMO KYC 团队)
- 审核期间收 1-2 封确认邮件 + 偶尔电话(确认業務的真实性,**不要慌**,正常说"开源软件订阅托管"即可)

### Step 4.0.7 收契约 + 住所通知

- 通过后收 PDF:**契約書 + 利用承諾書**
- **保存这两份 PDF** — 后续 開業届 / 銀行 / Paddle 都要

得到的对外公开信息:
- 住所:`東京都〇〇区〇〇 X-X-X (虚拟办公室)`(具体地址 GMO 通知)
- 电话(可选):若没勾电话转送,自己用 SMARTalk 050 番号

## 5. 配套:SMARTalk 050 番号(¥0/月)

### 为什么需要

特商法 + 開業届 都要电话号码。**不能用個人手机**(同住址公开问题)。

GMO オフィスサポート 默认不含电话(电话转送是另一个 plan,贵)。最便宜的解 = 自己用 050 IP 电话。

1. App Store / Google Play 搜「SMARTalk」 下载
2. 楽天 ID 登录(没有则现注册,免费)
3. 选 050 番号(自动分配,部分可选好记号)
4. 通话设置 + 留言
5. **立即可用**(无审核)

### 月费

- 月固定费:**¥0**
- 通话费:¥8.8/30 秒(很少有客户打电话)
- 接听:免费

### 这个号码用作

- 開業届 電話番号
- 銀行口座申请電話番号
- Paddle / Lemon Squeezy 商户电话
- 特商法表記 公开电话

## 8. 常见疑问

**Q1:GMO 审核会查我业务真实性吗?**

会,但不深。他们关心:
- 你不是为了诈骗 / 洗钱注册虚拟办公室
- 業種"软件/IT サービス" 是合法业务
- 不会问你具体卖什么(只要不写 VPN/翻墙等敏感词)

**Q2:邮件来了我怎么知道?**

- GMO 后台 dashboard 显示"邮件到达通知"
- 月转送 plan:GMO 自动月初寄到你個人住所
- 重要邮件可付 ¥330 即时转送

**Q3:税务署寄来通知 / 銀行寄来卡片能收到吗?**

✅ 可以,GMO 是合法地址(不是 PO Box),所有挂号 / 普通邮件都能签收。

**Q4:Paddle 审核会去日本实地查吗?**

不会。Paddle 在线审核,看你提供的開業届 + 銀行 + 网站住所一致性即可。

**Q5:特商法表記 必须用日文吗?**

国内卖给日本人:必须日文版
对外卖给海外:英文版 OK,但**有一份日文版 backup** 更稳

**Q6:有客户实地来访拜访 GMO 拠点怎么办?**

GMO 接待人员会:
- 确认对方身份
- 转告你(联系方式留你 050 + 邮箱)
- 不会让对方知道你是虚拟办公室用户

但**0-200 用户阶段几乎没人会专程来日本登门**(中国客户来不了日本,海外客户走 Twitter/邮件)。

**Q7:违约 / 解約怎么处理?**

- 月费契约,1 个月预告解約
- 解約金 ¥0(GMO 没违约金)
- 解約后 1 个月 GMO 仍代收邮件 → 再 30 天后停止

**Q8:可以同时注册多家虚拟办公室吗?**

可以,但浪费钱。一家够了。
