# 開業届 提交指南(個人事業主 / Nilou Network)

> 状态:Step 4 of payment merchant application playbook
> 适用:在日本提交開業届(個人事業主届出書),为申请 Paddle / Lemon Squeezy / 屋号付き口座做准备
> 预计耗时:在线生成 30 分钟 + 提交 5 分钟
> ⚠️ **前置 Step 4.0**:必须先办虚拟办公室 + 050 番号(否则個人住址 + 私人手机会因特商法被永久公开)— 见 `HOW-TO-RENT-VIRTUAL-OFFICE-JP.md`
> 关联:`docs/launch/PAYMENT-DECISION-FINAL-JP.md` §2.0、`PAYMENT-REALITY-CHECK.md` §7

---

## 1. 什么是開業届

**正式名称**:個人事業の開業・廃業等届出書(個人事業主 sole proprietor business commencement notification)

**作用**:
- 把你正式注册为日本個人事業主(无需公司注册)
- 让你能合法用屋号(`Nilou Network`)开业
- 申请 Paddle / Lemon Squeezy 等需要"日本個人事業主身份证明"时使用
- 为后续屋号付き銀行口座、青色申告控除做前置

**法律期限**:
- 開業日起 1 个月内提交(超期不罚款,但不能享青色申告控除)
- 过期可以补提,只是错失年度税务优惠

**费用**:¥0(完全免费)

## 2. 工具选择

三种主流方式,按推荐度:

### A. **Freee 開業届**(推荐,在线生成 + 引导式)

- 网址:`freee.co.jp/kaigyou/`
- 完全免费
- 在线问答式,30 分钟生成 PDF + 一键 e-Tax 提交
- 自动配套生成「青色申告承認申請書」(节税利器,选了就能享 ¥65万 控除)
- 支持英文界面(部分),但日文为主

### B. **Money Forward 開業届**

- 网址:`biz.moneyforward.com/kaigyou/`
- 完全免费
- 体验类似 Freee,稍偏会计软件入口
- 适合后续走 Money Forward 確定申告

### C. **手动下载 PDF + 邮寄**(不推荐)

- 国税庁官网下载空白表格
- 手填后邮寄/亲送税务署
- 工作量大,不适合首次提交

**我推荐 Freee** — 引导式 + 自动生成青色申告 + 提交 e-Tax 一站式。

## 4. 关键字段填写(过审核心)

### 4.1 屋号(やごう / trade name)

```
屋号: Nilou Network
```

**注意**:
- 屋号可以是英文(`Nilou Network`)
- 也可以加假名标注:`Nilou Network(ニルー・ネットワーク)`
- 后续屋号付き銀行口座、Paddle 申请、確定申告都用这个

### 4.2 事業の概要(关键!)

**不能写**:
- VPN サービス
- 翻墙 / プロキシ / アクセス回避
- 中国向け匿名化サービス
- 检 censorship 绕过

**应写(过审话术,英文 + 日文)**:

英文(Paddle 申请用):
> Nilou Network operates a managed hosting and subscription platform for the open-source Marzneshin panel. Customers subscribe to receive a managed instance of the open-source software, including infrastructure maintenance, technical support, SSL certificate management, and operator services.

日文(開業届 + 確定申告 用):
> 当事業は、オープンソースソフトウェア「Marzneshin」のマネージド・ホスティング及びサブスクリプション・プラットフォームを運営する。顧客はオープンソフトウェアの運用済みインスタンス、インフラ保守、技術サポート、運用代行を含むサブスクリプションを購読する。

**核心叙事**:
- 你卖的是"开源软件订阅托管服务"
- 客户买的是"managed instance + 技术支持 + 维护"
- 这是 100% 合法 SaaS 业务,**与 VPN 无关**(法律意义上 — 你是软件托管商,不是网络服务提供商)

### 4.3 業種(關鍵!)

**正確选项**(让後續 Paddle / 銀行 / 税务對齊):

| 業種コード | 名稱 | 用途 |
|---|---|---|
| **6201** | ソフトウェア業 | 推荐 #1 |
| **6202** | 情報処理・提供サービス業 | 推荐 #2 |
| 6203 | インターネット附随サービス業 | 备选(可能让 Paddle 联想到网络服务,慎用)|

**不能选**:
- 5990 通信業(會被 Paddle 怀疑為 VPN)
- 7299 その他の事業サービス業(模糊)
- 9499 他に分類されないサービス業(更模糊)

### 4.4 青色申告(必勾)

Freee / Money Forward 会自动同步生成「青色申告承認申請書」。

**好处**:
- 年度可享 ¥65万 控除(节税效果显著)
- 純損失 3 年内可结转
- 家族支付的报酬可计入経費

**条件**:
- 開業届 提交時同时申请(过期就要等下一年)
- 必须做复式簿記(Freee 自动做)

**强烈建议勾选**。

## 5. Step-by-step Freee 開業届 操作

### Step 1:进入 Freee

打开 `freee.co.jp/kaigyou/` → 点「無料で書類を作成する」

### Step 2:登录 / 注册

- 用 Google / Apple / Email 注册
- 邮箱用日本邮箱(你后续 Paddle / 銀行 都会收信)

### Step 3:回答引导式问题(约 20 分钟)

Freee 会按顺序问:
1. **氏名** — 你的全名(汉字 + 假名)
2. **生年月日** — 出生年月日
3. **住所** — 住民票上的住所(日文)
4. **マイナンバー** — 12 位
5. **電話番号** — 日本手机
6. **開業日** — 推荐今天或最近某天
7. **屋号** — `Nilou Network`
8. **事業の概要** — 用 §4.2 日文版
9. **業種** — 选 6201 ソフトウェア業
10. **青色申告** — 勾选 「申請する」
11. **従業員** — 0 人(個人单干)
12. **配偶者・親族** — 否(假設你目前是单干)

### Step 4:确认 PDF 预览

- Freee 自动生成两份 PDF:
  - 個人事業の開業・廃業等届出書
  - 所得税の青色申告承認申請書

### Step 5:提交方式

**A. e-Tax(电子提交,推荐,1 分钟)**:
- 需要 マイナンバーカード + IC 卡读取器(或手机 NFC)
- Freee 在线一键提交
- 即时受领,无需打印

**B. 邮寄**:
- 打印 PDF → 签字盖印章(認印 OK)→ 邮寄到管辖税务署
- 找你的管辖税务署:`https://www.nta.go.jp/about/organization/access/map.htm`
- 寄出后 1-2 周收到税务署回执

**C. 亲送税务署**:
- 打印 PDF → 带身份证 → 直接到税务署窗口
- 当场盖章受领,即时完成

**推荐 A(e-Tax)**:最快,Freee 引导无障碍。

### Step 6:保留受领证明

- e-Tax 提交后:下载受信通知 PDF,保存
- 邮寄/亲送:保留税务署回执

**这是后续 Paddle / 銀行 / 屋号付き口座 申请时的"個人事業主证明"**,务必保存好。

## 7. 常见疑问

**Q1:我在留资格是 工作签 / 经营管理 / 永住,可以申请 個人事業主 吗?**

- 永住 / 日本国籍 / 配偶者ビザ:✅ 任意业务
- 工作签(技术・人文知識・国際業務 / 高度人材):⚠️ 副業 OK,但 主业 需要"資格外活動許可"
- 经营管理ビザ:✅ 但通常已经有公司,個人事業主 是补充
- 留学 / 家族滞在:⚠️ 需要事先获取資格外活動許可

**对你**:看你目前在留资格,如不确定 → 入管 / 行政书士咨询(¥5000-10000)。

**Q2:営業日 / 開業日 怎么选?**

- 选你**实际开始为客户提供服务的日**
- 即使你已经在跑 nilou.cc 试运营,可以填**今天**作为正式開業日
- 太久之前的日子不要选(超过 1 年税务署可能要求说明)

**Q3:税务署管辖怎么找?**

- `https://www.nta.go.jp/about/organization/access/map.htm`
- 输入你的住址 → 自动查管辖税务署
- Freee 在线提交时自动选,无需手动查

**Q4:不交開業届 可以吗?**

- 法律层面:不强制(无罚款)
- 实操层面:**不交=不能享青色申告 ¥65万 控除 + 屋号付き口座申请受阻 + Paddle 申请缺合规材料**
- **推荐:交**(¥0 成本)

**Q5:屋号能后改吗?**

- 可以,但需要再交一份「変更届」
- 推荐一开始就定好(这就是为什么 `BRAND-NAMING-DECISION.md` SEALED 了 `Nilou Network` 后再交開業届)

**Q6:Freee 引导式过程中卡在某一项怎么办?**

- Freee 客服(中文/英文支持有限,日文为主)
- 行政書士(¥5000-15000 全程代办,适合初次或复杂情况)
- 我可以帮你看具体卡点,提供解读(虽然不能替你填)
