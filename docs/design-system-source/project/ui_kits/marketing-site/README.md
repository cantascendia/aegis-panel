# Marketing Site UI Kit

High-fidelity recreation of the **Nilou Network** landing site (`marketing/nilou-network/`).

## Components
- `Atoms.jsx` — `LotusMark`, `Eyebrow`, `Btn`, `Divider`, `FeatureIcon`
- `HeroSection.jsx` — `Topbar` (sticky glass), `Hero` (2-col, ripples + lotus breathe + dashboard mockup)
- `Sections.jsx` — `Features` (6-card grid), `Pricing` (4 plans, monthly/quarterly highlight)
- `FAQFooter.jsx` — `FAQ` (accordion w/ open state), `Footer` (4-col navy)

## Run
Open `index.html` directly. Loads React + Babel from CDN; uses `../../colors_and_type.css` and `../../assets/`.

## Voice notes
- Headlines are sentence-case Cormorant with an italic teal accent line ("Self-hosted apps, *without the self-hosting.*")
- All numbers carry caveats — "We target 99.9%" not "99.9% guaranteed"
- Plan tags use the gold pill on highlight cards, teal pill on standard
