## 2. SSH key 迁移（必做）

新电脑需要 `~/.ssh/id_ed25519`：

```bash
# 主机 A: 拷出私钥（**勿提交 git**）
cat ~/.ssh/id_ed25519
cat ~/.ssh/id_ed25519.pub

# 主机 B:
mkdir -p ~/.ssh && chmod 700 ~/.ssh
# 粘贴写入 ~/.ssh/id_ed25519，然后:
chmod 600 ~/.ssh/id_ed25519
ssh root@<vps-ip> echo ok
```

或在主机 B 生成新 ed25519，公钥追加到 VPS `~/.ssh/authorized_keys`。
