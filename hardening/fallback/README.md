# hardening/fallback/ — 备用通道(XHTTP / Hysteria2)

**状态**: ⏳ 待建(v0.3)

**为什么**: Reality 在部分国家/时段仍会被**启发式封锁**。需要自动回切到
基于 UDP 的 Hysteria2 或 HTTP/3 底层的 XHTTP。

## 方案

- 订阅链接同时下发 Reality + 备用协议,客户端按优先级自动试连
- Dashboard 客户端状态面板显示当前活跃协议 + 切换计数
- 告警:某节点备用协议使用率 > 30% 视为 Reality 被封征兆

## 前置

- v0.3 开始,早于此轮不做(上游无原生支持,引入会增加协议栈复杂度)
