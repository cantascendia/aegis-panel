Nilou Network について

Nilou Network は、日本の個人事業主が運営する、オープンソースソフトウェア向けのマネージドホスティング・サブスクリプションサービスです。お客様にはマネージド・デプロイメントをサブスクリプションでご提供し、テクニカルサポートとインフラ保守を当方が代行します。

当方が運用するソフトウェアは、寛容なオープンソースライセンスのもとで提供されています。ホストしているコンポーネントのソースコードは、関連するオープンソースライセンスに従い、サブスクライバーの方からのご請求に応じて提供します。

商標について: ブランド名「Nilou」はペルシャ語で「青蓮」を意味する一般語として使用しています。当事業は miHoYo / HoYoverse 及び『原神』フランチャイズとは一切関係ありません。
