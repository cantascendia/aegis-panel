事業者について

小さな事業者が、丁寧に運営しています。

Nilou Network は、日本の個人事業主が単独で運営する、オープンソースソフトウェア向けのマネージドホスティング・サブスクリプションサービスです。お客様にはマネージド・デプロイメントを月額・年額でご提供し、運用代行と技術サポートを当方がお引き受けします。

ホストしているソフトウェアコンポーネントは、寛容なオープンソースライセンスのもとで提供されています。ソースコードは関連ライセンスに従い、ご請求に応じて開示します。本ページではトラッキングスクリプトや解析ビーコン、第三者広告は使用していません。

商標について。 ブランド名「Nilou」はペルシャ語で「青蓮」を意味する一般的な単語として使用しています。Nilou Network は独立した事業であり、miHoYo / HoYoverse 及び『原神』フランチャイズとは関係ありません。
