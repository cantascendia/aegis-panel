よくある質問

ご検討の際によくいただくご質問にお答えします。

VPS とは何が違いますか?

VPS は素の Linux サーバーがお手元に届くだけで、インストール、パッチ適用、監視、バックアップは全てご自身で行う必要があります。Nilou Network は、構成済みのオープンソースアプリケーションを、メンテナンスとサポート付きでマネージド・インスタンスとしてご提供します。何かあった時にメールで連絡できる担当者がいます。

後でセルフホストに移行できますか?

はい。当方が運用しているソフトウェアはオープンソースですので、ご自身のハードウェアでも同じアプリケーションを構築できます。ご請求に応じて、データと設定を標準フォーマットで無償でエクスポートいたします。

返金ポリシーを教えてください

サブスクリプションはダッシュボードからいつでもキャンセルできます。未使用期間の返金は、ご承認後 7 営業日以内に処理いたします。詳細は特定商取引法に基づく表記をご確認ください。

バックアップとデータエクスポートはどう扱われますか?

日次スナップショットを 14 日間、週次スナップショットを 60 日間保持します。フルデータのエクスポートはいつでもご請求いただけ、標準フォーマット(SQL ダンプ、JSON、設定ファイル)で 3 営業日以内にお届けします。

どんな支払方法に対応していますか?

Alipay、WeChat Pay、USDT (TRC20)、主要クレジットカード(決済代行経由)。サブスクリプションはキャンセルされるまで自動更新されます。単発決済もご相談に応じます。

稼働率の目標はどのくらいですか?

マネージドインスタンスは月間 99.9% の稼働率を目標としています。現在の規模では契約上の SLA は提供していません — 守れない約束をするより、小さな事業であることを正直にお伝えする方を選びました。

標準カタログにないオープンソースアプリも運用できますか?

多くの場合、可能です。support@nilou.network まで、プロジェクトの URL とご利用目的をお知らせください。検討のうえ、適合する場合は初回セットアップ費用をお見積りします。

miHoYo や『原神』との関係はありますか?

ありません。ブランド名「Nilou」はペルシャ語で「青蓮」を意味する一般的な単語として使用しています。Nilou Network は独立した事業であり、miHoYo / HoYoverse 及び『原神』フランチャイズとは関係ありません。

小さな事業者が、丁寧に運営しています。

Nilou Network は、日本の個人事業主が単独で運営する、オープンソースソフトウェア向けのマネージドホスティング・サブスクリプションサービスです。お客様にはマネージド・デプロイメントを月額・年額でご提供し、運用代行と技術サポートを当方がお引き受けします。
