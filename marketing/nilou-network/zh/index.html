常见问题

我们听到最多的问题,诚实作答。

这和普通 VPS 有什么区别?

VPS 是一台空白的 Linux 服务器——安装、打补丁、监控、备份都得您自己做。Nilou Network 提供已配置好的开源应用托管实例,持续维护,出问题时有真人对接。

以后可以迁回自部署吗?

可以。我们运行的软件是开源的——您随时能在自己的硬件上部署同款应用。我们会按请求免费导出您的数据和配置,采用标准格式。

退款政策是怎样的?

订阅可在仪表盘内随时取消。未使用期间的退款,经核准后 7 个工作日内处理完毕。完整条款请见法律页面。

备份和数据导出怎么处理?

每日快照保留 14 天,每周快照保留 60 天。您可以随时申请完整数据导出——我们以标准格式(SQL 转储、JSON、配置文件)在 3 个工作日内交付。

支持哪些支付方式?

支付宝、微信支付、USDT(TRC20)、主流信用卡(经支付合作方)。订阅默认自动续费,可随时取消。也可按需安排单次付款。

可用性目标是多少?

托管实例月度可用性目标 99.9%。当前规模我们不提供合同 SLA——比起做不到的承诺,我们更愿意诚实告诉您这是一个小型事业。

能托管你们默认目录之外的开源应用吗?

大多数情况可以。请发邮件到 support@nilou.network,告诉我们项目 URL 和您的需求。评估后,如适合会报一次性配置费。

你们和米哈游或《原神》有关系吗?

没有。"Nilou" 在此为通用词汇使用,源自波斯语,意为"蓝睡莲"。Nilou Network 为独立事业,与米哈游 / HoYoverse 及《原神》系列无任何关联。

关于 Nilou Network

一个小事业者,用心经营。

Nilou Network 由日本独立个人事业主(個人事業主)运营,为开源软件提供托管订阅服务。客户订阅后获得托管部署、运营级支持,以及由我们代为完成的基础设施维护。

我们运行的软件均采用宽松的开源许可证。所托管组件的源代码可应请求提供,符合相关开源许可证条款。本页面不使用任何跟踪脚本、分析信标或第三方广告。
