关于 Nilou Network

Nilou Network 由日本个人事业主(個人事業主)运营,为开源软件提供托管订阅服务。客户订阅后获得托管部署、技术支持及由我们代为维护的基础设施。

我们运行的软件采用宽松的开源许可证。所托管组件的源代码可应订阅者请求提供,符合相关开源许可证要求。
