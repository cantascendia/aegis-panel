About Nilou Network

A small operation, run with care.

Nilou Network is operated as a Japanese sole-proprietor business (個人事業主) providing managed hosting subscriptions for open-source software. Customers subscribe for a managed deployment, receive operator-level support, and benefit from infrastructure maintenance handled on their behalf.

We host open-source software under permissive licenses. Source code for the components we operate is available on request, in accordance with the relevant open-source license terms. No tracking scripts, no analytics beacons, no third-party ads on this page.
